# Пример использования с Node.js

Этот пример представляет собой простой скрипт для Node.js,
который использует пакеты `@rci-tools/core` и `@rci-tools/adapter-axios`
для взаимодействия с RCI API. В примере показано, как:
- аутентифицироваться с помощью имени пользователя и пароля
- выполнить одиночный RCI-запрос
- выполнить несколько RCI-запросов в одном HTTP-запросе
- управлять "фоновыми процессами" (например, `tools ping` и `components list`)

## Необходимые условия

- Node.js (рекомендуется LTS-версия >= 22)
- Менеджер пакетов pnpm
- Устройство Keenetic/Netcraze, доступное в вашей сети

## Установка

1. **Перейдите в директорию проекта:**
   ```bash
   cd examples/simple-nodejs-demo
   ```

2. **Установите зависимости:**
   ```bash
   pnpm install
   ```

## Запуск приложения

Запустите демо:

```bash
pnpm dev
```

Приложение запросит у вас:
- IPv4-адрес устройства (по умолчанию: `192.168.1.1`)
- Имя пользователя (по умолчанию: `admin`)
- Пароль

После успешной аутентификации вы увидите в консоли как результаты выполнения обычных RCI-запросы,
так и результаты работы фоновых процессов.
