# Local Monitor

Local Monitor — это пример терминального приложения, работающего с устройствами Keenetic/Netcraze.
Приложение построено с использованием [Ink](https://github.com/vadimdemedes/ink) и React.
Связь с устройствами осуществляется при помощи пакетов `@rci-tools/core` и `@rci-tools/adapter-axios`.

<img src="./local-monitor.svg"/>

После того как вы укажете учетные данные для нового устройства, это приложение будет опрашивать статус устройства
и отображать его в таблице всех отслеживаемых устройств.

Учетные данные устройств хранятся в базе данных SQLite (файл внутри папки приложения).

> [!IMPORTANT]
> Local Monitor — это приложение-пример, не стоит использовать его для чего-то серьезного.
> Введенные вами данные об устройстве хранятся в виде открытого текста.

### Необходимые компоненты

- [Node.js](https://nodejs.org/) v18 или новее (рекомендуется Node 22+)
- [pnpm](https://pnpm.io/)

### Установка зависимостей

```bash
pnpm install
```

### Режим разработки

```bash
pnpm dev
```

### Сборка для продакшена

```bash
pnpm build
```

### Запуск приложения (после выполнения скрипта `build`)

```bash
pnpm start
```

### Другие скрипты

- `clean` – Удалить директорию `dist/`.
- `fmt` – Отформатировать исходные файлы с помощью dprint.
